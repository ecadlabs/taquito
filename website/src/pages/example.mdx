# Live Code Examples

This page demonstrates live JavaScript code blocks that can be executed directly in the browser.

## Hello World

```js live
console.log("Hello, World!")
console.log("This code runs live in your browser!")
```

## Working with Variables

```js live
const name = "Developer";
const greeting = `Hello, ${name}!`;
console.log(greeting);

const numbers = [1, 2, 3, 4, 5];
const sum = numbers.reduce((a, b) => a + b, 0);
console.log("Sum of numbers:", sum);
```

## Using Taquito

```js live
// Check the balance of a Tezos address
const balance = await Tezos.tz.getBalance('tz1fJ4q2nCYxfwR7LPDccSXycQ6ugngUChzz');
console.log('Balance:', balance.toString());
```

```js live
// Get information about a Tezos block
const block = await Tezos.rpc.getBlockHeader();
console.log('Current block level:', block.level);
console.log('Block hash:', block.hash);
```

## Configuring a Signer

For transactions that require signing, configure a signer:

```js live
// Configure the signer for transaction operations
await configureSigner();

// Now you can perform signed operations
const amount = 1;
const address = 'tz1h3rQ8wBxFd8L9B3d7Jhaawu6Z568XU3xY';

console.log(`Transferring ${amount} êœ© to ${address}...`);
try {
  const op = await Tezos.contract.transfer({ to: address, amount: amount });
  console.log(`Waiting for ${op.hash} to be confirmed...`);
  await op.confirmation(1);
  console.log(`Operation injected: https://ghost.tzstats.com/${op.hash}`);
} catch (error) {
  console.log(`Error: ${error.message}`);
}
```
