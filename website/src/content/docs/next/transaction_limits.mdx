Developers may wish to set different limits for a transaction before broadcasting it, for example to give it a better chance to be included first or to prevent the transaction from being backtracked due to an insufficient storage limit.

## Setting Limits

You can set the limits manually or let `Tezos.estimate` calculate an estimate of the required fees:

To set the fees manually:
```typescript
const contract = await Tezos.wallet.at('contractAddress');
const op = await contract.methodsObject.simple_param(5).send({
  storageLimit: 800,
  gasLimit: 8000,
  fee: 800
});
```

To set the fees using `Tezos.estimate`:
```typescript
const contract = await Tezos.wallet.at('contractAddress');

const estimateOp = await contract.methodsObject.simple_param(5).toTransferParams({});
const { 
  gasLimit,
  storageLimit,
  suggestedFeeMutez 
} = await Tezos.estimate.transfer(estimateOp);

const op = await contract.methodsObject.simple_param(5).send({
  storageLimit: storageLimit,
  gasLimit: gasLimit,
  fee: suggestedFeeMutez
});
```
